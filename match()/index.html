JavaScript String Method: .match()

    Definition:
  • this method matches a string against a regular expression
  • this method returns an array with the matches.
  • this method returns null if no match is found

Syntax

string.match(match)

Parameters
Parameter Description
match Required.
The search value.
A regular expression (or a string that will be converted to a regular expression).
Return Value
Type Description
An array
or null 		An array containing the matches.
null if no match is found.
    Notes:
    • The Difference Between String match() and String search()
    • The match() returns an array of matches.
    • The search() method returns the position of the first match.
    Reference:
  • W3schools

Example

Code:

let text = "The rain in SPAIN stays mainly in the plain";
let result1 = text.match("ain"); // Search for "ain" using a string
let result2 = text.match(/ain/); // Search for "ain" using a regular expression
let result3 = text.match(/ain/g); // Global search for "ain"
let result4 = text.match(/ain/gi); // Global case-insensitve, search for "ain"

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = result3;
document.getElementById ("example4").innerHTML = result4;
