JavaScript String Method: .match()

    Definition:
  • this method matches a string against a regular expression
  • this method returns an array with the matches.
  • this method returns null if no match is found

Syntax

string.match(match)

Parameters
Parameter Description
match Required.
The search value.
A regular expression (or a string that will be converted to a regular expression).
Return Value
Type Description
An array
or null 		An array containing the matches.
null if no match is found.
    Notes:
    • The Difference Between String match() and String search()
    • The match() returns an array of matches.
    • The search() method returns the position of the first match.
