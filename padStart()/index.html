JavaScript String Method: .padStart()

    Definition:
  • this method pads the string from the start
  • this method pads a string with another string (multiple times) until it reaches a given length.

Syntax

string.padStart(Length, string)

Parameters
Parameter Description
length Required.
The length of the resulting string.
string Optional.
The string to pad with.
Default is space.
Return Value
Type Description
A string A String of the specified length, with the padding applied from the start.
    Notes:
  • The padStart() method is a string method.
    To pad a number, convert the number to a string first.
  • padStart() is an ECMAScript7 (ES&) feature.
  • ES7 (JavaScript 2016) is supported in all modern browsers.
  • padStart() is not supported in internet Explorer or Edge 17 (or earlier).
    Reference:
  • W3schools

Example

Code:

let text1 = "1";
text1 = text1.padStart(4,"0");

document.getElementById ("example1").innerHTML = text1;

let text2 = "2";
document.getElementById ("example2").innerHTML = text2.padStart(4, "x");

let numb = 3;
let text3 = numb.toString();
document.getElementById ("example3").innerHTML = text3.padStart(4, 0);
