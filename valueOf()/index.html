JavaScript String Method: valueOf()

    Definition:
  • this method returns the primitive value of a string.
  • this method does not change the original string.
  • this method can be used to convert a string object into a string.

Syntax

string.valueOf()

Similar to:

  • toString()
Parameters
Parameter
NONE
Return Value
Type Description
A string The primitive value of the string.
    Notes:
  • The valueOf() method is the default method for JavaScript strings.
  • It is used internally by JavaScript
  • Normally, you will not use it in your code.
  • ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = "Hello World!";
let result1 = text1.valueOf();

document.getElementById ("example1").innerHTML = result1;

let text2 = "Hello World!";
let result2 = text2;

document.getElementById ("example2").innerHTML = result2;

let text3 = new String ("Hello World!");
let result3 = text3.valueOf()

document.getElementById ("example3").innerHTML = result3;
