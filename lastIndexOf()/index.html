JavaScript String Method: lastIndexOf()

    Definition:
  • this method returns the index (position) of the last occurence of a specified value in a string.
  • this method searches the string from the end to the beginning.
  • returns the index from the beginning (position 0).
  • returns -1 if the value is not found.
  • this method is case sensitive.

Syntax

string.lastIndexOf (searchvalue, start)

Parameters
Parameter Description
searchvalue Required.
The string to search for.
start Optional.
The position where to start.
Default value is string length.
Return Value
Type Description
A number The position where the search-value occurs.
-1 if it never occurs.
    Notes:
  • lastIndexOf() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text = "Hello planet earth, you are a great planet.";

let result1 = text.lastIndexOf("planet");
let result2 = text.lastIndexOf("Planet");
let result3 = text.lastIndexOf("planet", 20);

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = result3;
