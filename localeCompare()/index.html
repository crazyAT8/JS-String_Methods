JavaScript String Method: .localeCompare()

    Definition:
  • this method compares two strings in the current locale.
  • this method returns sort order -1, 1, or 0 (for before, after, or equal).
  • The current locale is based on the language setting of the browser.

Syntax

string.localeCompare (compareString)

Parameters
Parameter Description
compareString Required.
The string to compare with.
Return Value
Type Description
A number One of 3 values:
  1. -1 if the string is sorted before the compareString
  2. 0 if the two strings are equal
  3. 1 if the string is sorted after the compareString
    Notes:
  • localeCompare() is a ECMAScript (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = "ab";
let text2 = "cd";
let text3 = "ab";
let text4 = "A";
let text5 = "a";

let result1 = text1.localeCompare(text2);
let result2 = text2.localeCompare(text1);
let result3 = text1.localeCompare(text3);
let result4 = text4.localeCompare(text5);

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = result3;
document.getElementById ("example4").innerHTML = result4;
