JavaScript String Property: length

    Definition:
  • this property returns the length of a string.
  • The length property of an empty string is 0.

Syntax

string.length

Return Value
Type Description
A number The length of the string.
    Notes:
  • length is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers.
    Reference:
  • W3schools

Example

Code:

let text = "Hello World!";
let length = text.length;

document.getElementById ("example1").innerHTML = length;
