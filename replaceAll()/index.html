JavaScript String Method: .replaceAll()

    Definition:
  • this method searches a string for a calue or a regular expression.
  • this method returns a new string with all values replaced.
  • this method does not change the original string.
  • this method was introduced in JavaScript 2021.
  • this method does not work in Internet Explorer.

Syntax

string.replaceAll(searchValue, newValue)

Parameters
Parameter Description
searchValue Required.
The value, or regular expression, to search for.
newValue Required.
The new value (to replace with).
This parameter can be a JavaScript function.
Return Value
Type Description
A string A new string where the search values has been replaced.
    Notes:
  • If the parameter is a regular expression, the global flag (g) must be set, otherwise a TypeError is thrown.
    Reference:
  • W3schools

Example

Code:

// String
let text1 = "I love cats. Cats are very easy to love. Cats are very popular."
text1 = text1.replaceAll("Cats","Dogs");
text1 = text1.replaceAll("cats","dogs");

document.getElementById ("example1").innerHTML = text1;

// Regular Expression
let text2 = "I love cats too. Cats are very easy to love as well. Cats are popular";
text2 = text2.replaceAll (/Cats/g,"Dogs");
text2 = text2.replaceAll (/cats/g,"dogs");

document.getElementById ("example2").innerHTML = text2;

// Global, case-insensitive replacement
let text3 = document.getElementById ("example3").innerHTML;
let result3 = text3.replaceAll (/blue/gi, "red");
document.getElementById ("example3").innerHTML = result3;

// A function to return the replacement text
let text4 = document.getElementById ("example4").innerHTML;
let result4 = text4.replaceAll (/green|house|car/gi, function (x) {
    return x.toUpperCase();
});

document.getElementById ("example4").innerHTML = result4;

    Result:
  3. Mr Blue has a blue house an a blue car.
  4. Mr Green has a green house an a green car.
