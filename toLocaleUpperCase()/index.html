JavaScript String Method: .toLocaleUpperCase()

    Definition:
  • this method converts a string to uppercase letters, using current locale.
  • The locale is based on the language settings of the browser.
  • this method does not change the original string.
  • returns the same result as toUpperCase(), except for locales that conflict with the regular Unicode case mappings (such as Turkish).

Syntax

string.toLocaleUpperCase()()

Similar to:

  • toLowerCase()
  • toUpperCase()
  • toLocaleLowerCase()
Parameters
Parameter
NONE
Return Value
Type Description
A string A new string converted to uppercase according to current locale.
    Notes:
  • toLocaleUpperCase() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers

  • W3schools

Example

Code:

let text = "Hello World!";
let result = text.toLocaleUpperCase();

document.getElementById ("example1").innerHTML = result;
