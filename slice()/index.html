JavaScript String Method: .slice()

    Definition:
  • this method extracts a part of a string.
  • this method returns the extracted part in a new string.
  • this method does not change the original string.
  • The start and end parameters specifies the part of the string to extract.
  • The first position is 0, the second is 1, ...
  • A negative number selects from the end of the string.

Syntax

string.slice(start, end)

Similar to:

  • .split()
  • .substr()
  • .substring()
Parameters
Parameter Description
start Required.
The start position.
(First character is 0).
end Optional.
The end position (up to, but not including).
Default is string length.
Return Value
Type Description
A string The extracted part of the string.
    Notes:
  • slice() is an ECMAScript1 (ES1) feature
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = "Hello world!";

let result1 = text1.slice(0, 5); // First word
let result2 = text1.slice(3); // whole sting except for the first three letters
let result3 = text1.slice(3, 8);
let result4 = text1.slice(0, 1); // First character
let result5 = text1.slice(-1); // Last character
let result6 = text1.slice(0); // Whole string

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = result3;
document.getElementById ("example4").innerHTML = result4;
document.getElementById ("example5").innerHTML = result5;
document.getElementById ("example6").innerHTML = result6;
