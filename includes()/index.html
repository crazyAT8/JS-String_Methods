JavaScript String Method: .includes

    Definition:
  • this method returns true if a string contains a specified string.
  • Otherwise it returns false
  • this method is case sensitive

Syntax

string.includes (searchvalue, start)

Parameters
Parameter Description
searchvalue Required.
The string to search for.
start Optional.
The position to start from.
Default value is 0.
Return Value
Type Description
A boolean. true if the string contains the value, otherwise false.
    Notes:
  • includes() is an ECMAScript6 (ES6) feature.
  • ES6 (JavaScript 2015) is upported in all modern browsers.
  • not supported in Internet Explorer 11 (or earlier).
    Reference:
  • W3schools

Example

Code:

let text = "Hello world, welcome to the universe.";
let result1 = text.includes("world");
let result2 = text.includes("world", 12);

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
