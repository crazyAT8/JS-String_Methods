JavaScript String Method: endsWith()

    Definition:
  • this method returns true if a string ends with a specified string.
    otherwise it returns false
  • this method is case sensitive

Syntax

string.endsWith(searchvalue, Length)

Parameters
Parameter Description
searchvalue Required.
The string to search for.
length Optional.
The length of the string to search.
Default value is the length of the string.
Return Value
Type Description
A boolean true if the string ends with the value, otherwise false.
    Notes:
  • is an ECMAScript6 (ES6) feature
  • ES6 (JavaScript 2015) is supported in all modern browsers
  • not supported in Internet Explorer 11 (or earlier)
