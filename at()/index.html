JavaScript String Method: .at()

    Definition:
  • this method returns an indexed character from a string.
  • this method returns the same as [ ].

Syntax

string.at(index)

Parameters
Parameter Description
A String Optional.
The index (position) of the character to be returned.
Default is 0.
-1 returns the last character.
Return Value
Type Description
A String A single character.
The character in the given position (index) in the string.
    Notes:
  • supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text = "MotherPussBucket";
let character = text.at(2);
let character2 = text[1];
let character3 = text.at();
let character4 = text.at(-1);

document.getElementById("example1").innerHTML = character;
document.getElementById("example2").innerHTML = character2;
document.getElementById("example3").innerHTML = character3;
document.getElementById("example4").innerHTML = character4;
