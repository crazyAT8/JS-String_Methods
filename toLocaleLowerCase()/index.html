JavaScript String Method: .toLocaleLowerCase()

    Definition:
  • this method converts a string to lowercase letters, using current locale.
  • The locale is based on the language settings of the browser.
  • this method does not change the original string.
  • returns the same result as toLowerCase(), except for locales that conflict with the regular Unicode case mappings (such as Turkish).

Syntax

string.toLocaleLowerCase()()

Similar to:

  • toLowerCase()
  • toUpperCase()
  • toLocaleUpperCase()
Parameters
Parameter
NONE
Return Value
Type Description
A string A new string converted to lowercase according to current locale.
    Notes:
  • toLocaleLowerCase() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text = "Hello World!";
let result = text.toLocaleLowerCase();

document.getElementById ("example1").innerHTML = result;
