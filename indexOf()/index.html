JavaScript String Method: .indexOf()

    Definition:
  • this method the position of the first occurrence of a value in a string.
  • this method returns -1 if the value is not found.
  • this method is case sensitive.

Syntax

string.indexOf (searchvalue, start)

Parameters
Parameter Description
searchvalue Required.
The string to search for.
start Optional. The position to start from (default is 0).
Return Value
Type Description
A number The first position where the search-value occurs.
-1 if it never occurs.
    Notes:
    • The differense Btw String indexof() and string search()
    • the indexOf() method cannot search against a regular expression.
    • the search() cannot take a start position argument.
  • indexOf() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported inall browsers

Example

Code:

let text = "Hello world, welcome to the universe.";
let result1 = text.indexOf("welcome");
let result2 = text.indexOf("Welcome");

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = text.indexOf("e");
document.getElementById ("example4").innerHTML = text.indexOf("e", 5);
document.getElementById ("example5").innerHTML = text.indexOf("a", 5);
