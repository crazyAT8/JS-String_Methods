JavaScript String Method: .fromCharCode()

    Definition:
  • this method converts Unicode values tocharacters
  • is a static method of the String object.

Syntax

String.fromCharCode(n1, n2, ..., nX)

Parameters
Parameter Description
n1, n2, nX Required.
One. or more Unicode values to be converted.
Return Value
Type Description
A String A string representing the unicode character(s)
    Notes:
  • is an ECMAScript1 (ES1) feature
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools
  • Complete Unicode reference

Example

Code:

let text = String.fromCharCode(65);
let text2 = String.fromCharCode(72, 69, 76, 76, 79);

document.getElementById ("example1").innerHTML = text;
document.getElementById ("example2").innerHTML = text2;
