JavaScript String Method: .trim()

    Definition:
  • this method removes whitespace from both sides of a string.
  • this method does not change the original string.

Syntax

string.trim()

Similar to:

  • .trimEnd()
  • .trimStart()
  • .padEnd()
  • .padStart()
Parameters
Parameter
NONE
Return Value
Type Description
A string A string with removed whitespace from both ends.
    Notes:
  • .trim() is an ECMAScript5 (ES5) feature.
  • ES5 (JavaScript 2009) fully supported in all modern browsers since July 2013:
    Reference:
  • W3schools

Example

Code:

let text = " Hello World! ";
let result = text.trim();

document.getElementById ("example1").innerHTML = text;
document.getElementById ("example2").innerHTML = result;
