JavaScript String Method: .replace()

    Definition:
  • this method searches a string for a value or a regular expression.
  • this method returns a new string with the value(s) replaced.
  • this method does not change the original string

Syntax

string.replace(searchValue, newValue)

Parameters
Parameter Description
searchValue Required.
The value, or regular expression, to search for.
newValue Required.
The new value (to replace with).
Return Value
Type Description
A string A new string where the specified value(s) has been replaced.
    Notes:
  • If you replace a value, only the first instance will be replaced. To replace all instances, use a regular expression with the g modifier set.
  • replace() is an ECMAScript (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = document.getElementById ("example1").innerHTML;
document.getElementById ("example1").innerHTML = text1.replace("Arizona", "Nebraska")

// A global replacement
let str = document.getElementById ("example2").innerHTML;
let res = str.replace (/blue/g, "red");
document.getElementById ("example2").innerHTML = res;

// A global, case-insensitive replacement
let text2 = document.getElementById ("example3").innerHTML;
let result1 = text2.replace (/blue/gi, "red");
document.getElementById ("example3").innerHTML = result1;

// A function to return the replacement text
let text3 = document.getElementById ("example4").innerHTML;
let result2 = text3.replace (/blue|house|car/gi, function (x) {
    return x.toUpperCase();
});

document.getElementById ("example4").innerHTML = result2;

    Result:
  1. Visit Arizona
  2. Mr Blue has a blue house and a blue car.
  3. Mr Blue has a blue house and a blue car.
  4. Mr Blue has a blue house and a blue car.
