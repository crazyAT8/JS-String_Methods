JavaScript String Method: .substring()

    Definition:
  • this method extracts characters, between two indices (positions), from a string, and returns the substring.
  • this method extracts characters from start to end (exclusive).
  • this method does not chan ge the original string.
  • If start is greater than end, arguments are swapped: (4, 1) = (1, 4).
  • Start or end values less than 0, are treated as 0.

Syntax

string.substring(start, end)

Similar to:

  • .split() Method
  • .slice() Method
  • .substr() Method
Parameters
Parameter Description
start Required.
Start position.
First character is at index 0.
end Optional.
End position (up to, but not including). If omitted: the rest of the string.
Return Value
Type Description
A string A string containing the extracted characters.
    Notes:
  • substring() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) if fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = "Hello World!";

let result1 = text1.substring(1, 4);
let result2 = text1.substring(2);
let result3 = text1.substring(4, 1);
let result4 = text1.substring(-1);
let result5 = text1.substring(0, 1);
let result6 = text1.substring(text1.length - 1);

document.getElementById ("example1").innerHTML = result1;
document.getElementById ("example2").innerHTML = result2;
document.getElementById ("example3").innerHTML = result3;
document.getElementById ("example4").innerHTML = result4;
document.getElementById ("example5").innerHTML = result5;
document.getElementById ("example6").innerHTML = result6;
