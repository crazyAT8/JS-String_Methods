JavaScript String Method: .toLowerCase()

    Definition:
  • this method converts a string to lowercase letters.
  • this method does not change the original string.

Syntax

string.toLowerCase()

Similar to:

  • toUpperCase()
  • toLocaleLowerCase()
  • toLocaleUpperCase()
Parameters
Parameter
NONE
Return Value
Type Description
A string The string converted to lowercase
    Notes:
  • toLowerCase() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text = "Hello World!";
let result = text.toLowerCase();

document.getElementById ("example1").innerHTML = result;
