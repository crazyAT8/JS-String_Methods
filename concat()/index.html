JavaScript String Method: .concat()

    Definition:
  • this method joins two or more strings.
  • this method does not change the existing strings.
  • this method returns a new string.

Syntax

string.concat(string1, string2, ..., stringX)

Parameters
Parameter Description
string1,
string2,
...
stringX		 Required.
The strings to be joined.
Return Value
Type Description
A string A new string containing the combined strings.
    Notes:
  • concat() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text1 = "Hello";
let text2 = "world!";
let text3 = "Have a nice day!";
let result = text1.concat(" ", text2, " ", text3);

document.getElementById("example1").innerHTML = result;
