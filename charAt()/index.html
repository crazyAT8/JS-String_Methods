JavaScript String Method: .charAt()

    Definition:
  • this method method returns the character at a specified index (position) in a string.

Syntax

string.chatAt(index)

Parameters
Parameter Description
index Optional.
The index (position) of the character.
Default = 0.
Return Value
Type Description
String The character at the specified index. Empty string ("") if the index is out of range.
    Notes:
  • charAt() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers
    Reference:
  • W3schools

Example

Code:

let text = "hello world";
let letter = text.charAt(3);
let letter2 = text.charAt(text.length-1);
let letter3 = text.chatAt(15);
let letter4 = text.charAt();


document.getElementById ("example1").innerHTML = letter;
document.getElementById ("example2").innerHTML = letter2;
document.getElementById ("example3").innerHTML = letter3;
document.getElementById ("example4").innerHTML = letter4;
