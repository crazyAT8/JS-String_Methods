JavaScript String Method: .toString()

    Definition:
  • this method returns a string as a string.
  • this method does not change the original string.
  • this method can be used to convert a string object into a string.

Syntax

string.toString()

Similar to:

  • valueOf()
Parameters
Parameter
NONE
Return Value
Type Description
A string The content of the string.
    Notes:
  • Every JavaScript object object has a toString() method.
  • The toString() method is used internally by JavaScript when an object needs to be displayed as a text (like in HTML), or when an object needs to be used as a string.
  • Normally, you will not use it in your own code.
  • toString() is an ECMAScript1 (ES1) feature.
  • ES1 (JavaScript 1997) is fully supported in all browsers.
    Reference:
  • W3schools

Example

Code:

let text = "Hello World!";
let result = text.toString();

document.getElementById ("example1").innerHTML = result;

let text2 = new String ("Hello World!");
let result2 = text2.toString();

document.getElementById ("example2").innerHTML = result2;
